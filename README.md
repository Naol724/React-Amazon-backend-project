# Ecommerce Website Backend API

Express.js backend for Ecommerce Website with Stripe payment integration.

## Deployment on Render

### Environment Variables
Set these in Render dashboard:
- `STRIPE_SECRET_KEY`: Your Stripe secret key

### Build Command
```bash
npm install
```

### Start Command
```bash
npm start
```

### Port
The app runs on port 5000 by default. Render will provide the PORT environment variable.

## Local Development
```bash
npm install
npm start
```

Server runs on http://localhost:5000
